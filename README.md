# Ribon Monitoring
Proof of concept for monitoring of the Ribon DApp Environment:
[ribonstatus.herokuapp.com/](ribonstatus.herokuapp.com/)
<hr/>

## Features
- Principal DApp Services monitoring
- Cloud monitoring (AWS)
- 2 types of response (Web or JSON)
- Caching
